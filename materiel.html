Raspberry Pi 4 Model B

Pourquoi? Le Raspberry Pi 4 Model B est choisi pour sa puissance de traitement et sa capacité à gérer des tâches complexes comme la capture d'images et la gestion de données. Il est également compatible avec une large gamme de modules et d'accessoires, ce qui le rend idéal pour des projets personnalisés.

Raspberry Pi Official Camera Module V3

Pourquoi? Ce module de caméra est parfait pour la photographie en temps réel, l'enregistrement vidéo, la détection de mouvement et les applications de sécurité. Il est compatible avec le Raspberry Pi 4 Model B grâce à son connecteur CSI et offre une résolution photo maximale de 4608 x 2592 pixels, ce qui est idéal pour capturer des images de haute qualité des oiseaux

Micro SD Card

Pourquoi? La carte Micro SD est utilisée pour stocker les données capturées par la caméra, telles que les images et les vidéos. Elle permet de conserver les données sur le Raspberry Pi sans utiliser de stockage externe, ce qui est pratique pour les projets mobiles ou portables

Power Supply

Pourquoi? Le fournisseur d'alimentation est essentiel pour assurer que le Raspberry Pi fonctionne correctement. Un fournisseur d'alimentation de qualité garantit que le Raspberry Pi reçoit l'énergie nécessaire pour fonctionner sans interruption, ce qui est crucial pour la capture continue d'images ou de vidéos.

Blocs lego

Pourquoi? Nous avons besoin d'un support pour la caméra. Les blocs lego sont parfaits pour cela. Un des membres de l'équipe en à une tonne chez lui, donc c'est gratuit. Nous allons simplement construire une base pour que la caméra reste stable.
