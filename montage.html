Créer un système de détection d'oiseaux avec Raspberry Pi, OpenCV et envoi de photos en ligne

Le projet consiste à construire un appareil qui permettra au client de prendre des photos d'oiseaux dégustant de délicieuses graines dans sa mangeoire. Le client veut pouvoir prendre des photos de ses oiseaux sans avoir à être présent et pouvoir les regarder sur un site web par la suite.

Niveau de compétence requis : intermédiaire.

Liste des composants matériels nécessaires

  • Raspberry Pi 4
  • Caméra Raspberry Pi
  • Carte microSD
  • Adaptateur secteur USB-C
  • Boîtier pour Raspberry Pi
  • Support de caméra Raspberry Pi
  • Boîtier pour raspberry Pi

1. Configuration initiale de la Raspberry Pi

  1. Installation du système d'exploitation

    Téléchargez la dernière version de Raspbian depuis le site officiel de Raspberry Pi.

    Utilisez un utilitaire de votre choix pour flasher l'image sur une carte microSD.

    Insérez la carte microSD dans votre Raspberry Pi et démarrez-la.

  2. Configuration du système

    Suivez les instructions à l'écran pour terminer la configuration initiale de Raspbian, y compris la définition de la langue, du fuseau horaire et du mot de passe.

  3. Connexion à Internet

    Si vous utilisez une connexion Ethernet, la Raspberry Pi devrait se connecter automatiquement à Internet.

    Pour le Wi-Fi, cliquez sur l'icône Wi-Fi en haut à droite de l'écran et sélectionnez votre réseau. Entrez le mot de passe si nécessaire.

2. Installation d'OpenCV sur la Raspberry Pi

  1. Mise à jour du système

    Ouvrez un terminal et exécutez les commandes suivantes pour mettre à jour les packages du système :

    sudo apt update
    sudo apt upgrade

  2. Installation des dépendances

    Avant d'installer OpenCV, nous devons installer quelques dépendances. Exécutez les commandes suivantes dans le terminal :

    sudo apt install build-essential cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev libv4l-dev libjpeg-dev libpng-dev libtiff-dev libatlas-base-dev gfortran python3-dev python3-pip

  3. Téléchargement d'OpenCV

    Téléchargez la dernière version d'OpenCV à partir du référentiel GitHub officiel en utilisant Git :

    git clone https://github.com/opencv/opencv.git

  4. Compilation et installation d'OpenCV

    Naviguez vers le répertoire où vous avez téléchargé OpenCV, puis créez un répertoire build et accédez-y :

    cd opencv
    mkdir build
    cd build

    Configurez la compilation en utilisant CMake :

    cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local ..

    Compilez et installez OpenCV :

    make -j4
    sudo make install
    sudo ldconfig

  5. Vérification de l'installation

    Pour vérifier si l'installation a réussi, ouvrez un terminal et exécutez Python. Ensuite, importez OpenCV :

    python3
    import cv2
    cv2.__version__

    Si aucune erreur n'est affichée et que la version d'OpenCV est imprimée, cela signifie qu'OpenCV a été installé avec succès.

3. Capturer les images avec la caméra

  1. Utilisation de la bibliothèque picamera pour capturer des images depuis la caméra.

    Pour capturer des images avec la caméra de la Raspberry Pi, nous utiliserons la bibliothèque Python picamera.

    Voici un exemple de code pour capturer une image :

    from picamera import PiCamera
                      import time
                      
                      camera = PiCamera()
                      
                      # Capture de l'image
                      camera.capture('image.jpg')
                      
                      # Fermeture de la caméra
                      camera.close()

4. Traitement d'images avec OpenCV

  1. Chargement de l'image capturée dans OpenCV.

    import cv2
            
            # Charger l'image capturée
            image = cv2.imread('image.jpg')

  2. Utilisation d'un modèle pré-entraîné pour la détection d'oiseaux.

    OpenCV offre des modèles pré-entraînés pour la détection d'objets, y compris des oiseaux.

    Voici comment charger un modèle pré-entraîné pour la détection d'oiseaux :

    Après avoir détecté les oiseaux dans l'image, nous pouvons dessiner des rectangles autour d'eux pour les mettre en évidence.

    # Dessiner des rectangles autour des oiseaux détectés

5. Envoi des photos en ligne

  1. Configuration d'un service en ligne pour recevoir les photos (par exemple, un serveur web avec une API).

    Pour envoyer les photos capturées par la Raspberry Pi en ligne, nous devons configurer un service web avec une API qui peut recevoir les photos et les stocker sur un serveur distant.

    Vous pouvez utiliser des services cloud comme AWS, Google Cloud, ou mettre en place votre propre serveur web avec une API personnalisée.

  2. Utilisation de bibliothèques Python pour envoyer les photos au service en ligne.

    Une fois que vous avez configuré votre service en ligne, vous pouvez utiliser des bibliothèques Python pour envoyer les photos capturées par la Raspberry Pi au serveur distant.

    Voici un exemple d'utilisation de la bibliothèque requests pour envoyer une photo à un service web :

    import requests
            
            # Charger la photo capturée
            with open('oiseau_detecte.jpg', 'rb') as file:
                photo = file.read()
            
            # Envoyer la photo au service en ligne
            response = requests.post('https://votre-api.com/upload', files={'photo': photo})
            
            # Vérifier la réponse
            if response.status_code == 200:
                print("La photo a été envoyée avec succès !")
            else:
                print("Une erreur s'est produite lors de l'envoi de la photo.")
